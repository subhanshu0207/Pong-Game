#Pong Game

A simple Pong game made just with java Swing and AWT

##Features
-Paddle Movement
-Computer Paddle
-Score counting
-Ball physics 

###Controls
W / Up arrow key to move the paddle up
S / Down arrow key to move the paddle down

I know, nothing crazy.

####Problems
-Lags a bit mid-game
-The computer is too easy to predict
